# DiKeVoi Monorepo

This is the monorepo version for DiKeVoi.

# Run Frontend And Backend Concurrently

In the root directory, `concurrently` is installed for running both processes with one command. Use

```
pnpm start
```

to run both Frontend and Backend processes.

# Testing

This project uses **Jest** and **React Native Testing Library** for testing.

## Run tests locally

- **Run all tests:**
  ```bash
  npm test
  ```

- **Run tests with coverage report:**
  ```bash
  npm run test:coverage
  ```
  After running this command, you can view the detaied HTML report at:
  `coverage/lcov-report/index.html`

- **Watch mode:**
  ```bash
  npx jest --watch
  ```
